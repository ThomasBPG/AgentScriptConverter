Small pet project for migrating and converterting Salesforce GenAiPlannerBundle metadata to Agent Script format.
The script converts the XML-based GenAiPlannerBundle metadata structure to the new YAML-like Agent Script (.agent) format used by Agentforce Builder.

Build by vibecoding with OpenCode and Anthropic Claude Sonnet 4.5.

Please create pull requests with eventual issues you encounter.
